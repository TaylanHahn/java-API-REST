### Spring Web API
Desenvolvimento de uma API web simples, com o objetivo de estudar os conceitos fundamentais de uma API, bem como o uso prático de anotações do Spring Boot.

**Neste exercício foi abordando:**
- Organização de camadas
- Estruturação de controllers
- Documentação de API com *Swagger*
- Uso das *Exception Handlers* 
- Testes de requisição com Insomnia.
